# ADR-002: Technologie-Stack

## Entscheidung

- Backend: Python + FastAPI
- Frontend: Streamlit (MVP)
- DB: PostgreSQL
- Cache: Redis

---

## Begründung

- Python stark für Finance & AI
- FastAPI performant und modern
- Streamlit für schnelle UI-Entwicklung

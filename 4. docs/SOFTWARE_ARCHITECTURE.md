# Software Architecture

## Architekturmodell

### 1. Data Layer
- PostgreSQL
- Markt- und Nutzerdaten

### 2. Analysis Layer
- Berechnung von:
  - Rendite
  - Risiko
  - Diversifikation

### 3. AI Layer
- Interpretation der Daten
- Erklärungen
- Empfehlungen (nicht Handel!)

### 4. Cockpit Layer
- UI / Dashboard
- Visualisierung
